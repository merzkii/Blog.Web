# Blog.Web

###  рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃњрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃарЃЮрЃџрЃћрЃЉрЃў

рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃБрЃЎрЃЋрЃћ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ seed-рЃўрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃарЃЮрЃџрЃћрЃЉрЃў.

рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃерЃћрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ:

| рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў       | рЃърЃљрЃарЃЮрЃџрЃў      | рЃарЃЮрЃџрЃў    |
|-------------------|-------------|---------|
| `admin`       | `Admin123`  | Admin   |
| `user`     | `User123`   | User    |

> рЃљрЃа рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљрЃА.

---

### рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ

- FluentValidation рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ Application рЃцрЃћрЃюрЃљрЃерЃў рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.
- рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃЉрЃарЃБрЃюрЃЊрЃћрЃЉрЃљ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ `ApiErrorResponseDTO` рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў.
- MVC рЃцрЃћрЃюрЃљрЃерЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃљрЃЏ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ рЃЊрЃљ Bootstrap-рЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЕрЃћрЃюрЃљ.

---

### рЃљрЃърЃў рЃЊрЃљ Middleware

- рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ `ExceptionHandlingMiddleware`-рЃўрЃЌ.
- рЃАрЃбрЃљрЃбрЃБрЃА рЃЎрЃЮрЃЊрЃћрЃЉрЃў рЃЉрЃарЃБрЃюрЃЊрЃћрЃЉрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ (`400`, `401`, `404`, `500`) рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃЌ.
- рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃљрЃю рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА рЃЉрЃарЃБрЃюрЃЊрЃћрЃЉрЃљ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃњрЃќрЃљрЃЋрЃюрЃўрЃџрЃћрЃЉрЃў.

---

###  рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃърЃљрЃбрЃћрЃарЃюрЃћрЃЉрЃў

- рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ **Clean Architecture** рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃўрЃЌ:
  - **Domain**, **Application**, **Infrastructure**, **API**, рЃЊрЃљ **Web UI (MVC)** рЃцрЃћрЃюрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ.
- рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ **MediatR** CQRS рЃърЃљрЃбрЃћрЃарЃюрЃў.
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃ«рЃЊрЃћрЃЉрЃљ **Repository** рЃърЃљрЃбрЃћрЃарЃюрЃўрЃЌ.

---

## рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

- Role-based Access Control РђЊ Guest, User, Admin рЃарЃЮрЃџрЃћрЃЉрЃўрЃЌ

# рЃАрЃ«рЃЋрЃљ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў:
рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃЏрЃЮрЃюрЃЮрЃџрЃўрЃЌрЃБрЃарЃўрЃљ, рЃЌрЃБрЃЏрЃфрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ Clean Architecture-рЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў: Application, Domain, Infrastructure, Web/API рЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.

Web рЃърЃарЃЮрЃћрЃЦрЃбрЃў (MVC) рЃўрЃДрЃћрЃюрЃћрЃЉрЃА HttpClient-рЃА API-рЃАрЃЌрЃљрЃю рЃЊрЃљрЃАрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃџрЃљрЃЊ.

рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃЊрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ Exception Handling Middleware-рЃўрЃЌ API рЃцрЃћрЃюрЃљрЃерЃў.

Validation Error-рЃћрЃЉрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃљ рЃЋрЃћрЃџрЃћрЃЉрЃўрЃА рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ Bootstrap-рЃўрЃА рЃАрЃбрЃўрЃџрЃерЃў.

# рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ
рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ Solution Visual Studio 2022/2023-рЃерЃў.

рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ Blog.API рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃбрЃљрЃарЃбрЃљрЃъ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ.

рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃљрЃЊ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ Blog.Web MVC рЃърЃарЃЮрЃћрЃЦрЃбрЃў.

рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃАрЃўрЃЊрЃћрЃЉрЃў.

---

­Ъњг рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃю рЃњрЃљрЃАрЃљрЃарЃЎрЃЋрЃћрЃЋрЃљрЃЊ, рЃљрЃа рЃЏрЃЮрЃњрЃћрЃарЃўрЃЊрЃЮрЃЌ рЃЏрЃЮрЃЏрЃгрЃћрЃарЃћрЃЌ.
